# Search Workflow

Finds news related to user's TELOS interests and posts to Notion with source URLs.

## Execution Steps

### 1. Extract TELOS Topics

Read user's TELOS files to understand interests:

```bash
# Read key TELOS files
read ~/.claude/PAI/USER/TELOS/GOALS.md
read ~/.claude/PAI/USER/TELOS/PROBLEMS.md
read ~/.claude/PAI/USER/TELOS/PREDICTIONS.md
read ~/.claude/PAI/USER/TELOS/BELIEFS.md
read ~/.claude/PAI/USER/TELOS/CHALLENGES.md
```

**Extract these topic areas:**
- AI and economic disruption
- Community building and local resiliency
- Sustainability and supply chains
- Men's development and rites of passage
- Specific predictions being tracked
- Other interests mentioned in TELOS files

### 2. Build Search Queries

Create targeted search queries with date filter for last week:

**Query Format:**
```
"[topic keyword] [year]" site:[optional source]
```

**Example queries:**
- "AI job losses 2026"
- "community building initiatives 2026"
- "supply chain resilience 2026"
- "men's retreats rites of passage 2026"
- "[specific prediction topic] 2026"

**Date Strategy:**
- WebSearch returns recent results automatically
- Use current year (2026) in queries for currency
- Mention "last week" or "this week" in query when relevant

### 3. Execute Searches

Use WebSearch tool for each topic area:

```
WebSearch: "[topic] 2026"
```

**For each result:**
- Extract article title
- Extract source URL
- Note publication date if available
- Write 1-sentence summary of relevance

**Collect 3-5 articles per topic area.**

### 4. Group Results by Topic

Organize findings into categories:

```markdown
## AI & Economy
- [Title](URL) - Summary

## Community Building
- [Title](URL) - Summary

## Sustainability
- [Title](URL) - Summary

## Men's Development
- [Title](URL) - Summary

## Predictions Tracking
- [Title](URL) - Summary
```

### 5. Post to Notion

**Step 5a: Create the Notion page content**

Build markdown content with all articles grouped by topic:

```markdown
# Weekly TELOS News - [start date] to [end date]

*News related to your TELOS interests from the past week*

---

## AI & Economy

[Articles about AI disruption, job markets, economic transformation]

- [Article Title](source URL) - Brief description of relevance
- [Article Title](source URL) - Brief description

## Community Building

[Articles about community initiatives, social connection, local resiliency]

- [Article Title](source URL) - Brief description
- [Article Title](source URL) - Brief description

## Sustainability

[Articles about supply chains, energy, climate adaptation]

- [Article Title](source URL) - Brief description

## Men's Development

[Articles about mentorship, rites of passage, men's work]

- [Article Title](source URL) - Brief description

## Predictions Tracking

[Articles related to predictions in PREDICTIONS.md]

- [Article Title](source URL) - Brief description

---

*Generated by TelosNews skill on [date]*
```

**Step 5b: Create page with Notion MCP**

Use the `mcp__notion__notion-create-pages` tool:

```typescript
{
  "pages": [{
    "properties": {
      "title": "Weekly TELOS News - [date range]"
    },
    "content": "[full markdown content from 5a]"
  }]
}
```

**Important:**
- The `parent` parameter is optional - omitting it creates a workspace-level page
- User can organize it later
- Include full markdown content with all sections and links
- Ensure all URLs are properly formatted as `[text](url)`

### 6. Report Success

Output to user:

```
✅ Found [N] articles across [M] topic areas
📄 Posted to Notion: "Weekly TELOS News - [date range]"
🔗 View in Notion app or web

## Summary by Topic:
- AI & Economy: [N] articles
- Community Building: [N] articles
- Sustainability: [N] articles
- Men's Development: [N] articles
- Predictions Tracking: [N] articles
```

## Error Handling

**If WebSearch fails:**
- Try alternative query phrasing
- Continue with other topics
- Report which topics had issues

**If Notion fails:**
- Output the markdown content directly
- Provide manual instructions for posting
- Save content to temp file as backup

**If no results found:**
- Report which topics had no recent news
- Suggest trying different timeframe
- Still create page with "No recent news" sections

## Tips for Quality Results

1. **Query Specificity**: Use specific keywords from TELOS files, not generic terms
2. **Source Diversity**: Mix mainstream and specialized sources
3. **Relevance Check**: Only include articles directly relevant to TELOS topics
4. **URL Verification**: Ensure all URLs are valid and complete
5. **Summary Clarity**: Make summaries actionable - why does this matter to user's goals?

## Notes

- Skill runs on-demand (not scheduled)
- User triggers manually when they want news update
- Can be combined with other skills for deeper analysis
- Results are grouped for easy scanning by interest area
